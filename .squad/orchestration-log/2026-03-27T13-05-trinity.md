### Key Decisions Made

1. **Language:** Rust with edition 2021
   - Type safety and performance for compiler phases
   - Comprehensive error reporting with Span information

2. **Architecture:** Lexer → Token stream → Recursive descent parser → AST
   - Classic compiler pipeline matching specification
   - Single-error-and-stop strategy for now
   - All errors carry Span (line, column) metadata

3. **Struct Literal Ambiguity Resolution**
   - Pre-scan pass collects all struct/enum type names before parsing
   - When parser sees `Ident {`, checks if identifier is a known type
   - Disambiguates struct literal from block expression

4. **Assignment Detection**
   - Uses lookahead scanning past place expressions (`ident.field[index]...`)
   - Finds `=` before committing to assignment vs expression statement

5. **`fn!` Token Handling**
   - Lexer treats `fn!` as single `FnBang` token, not `fn` + `!`
   - Follows spec directive that `fn!` is a single token
